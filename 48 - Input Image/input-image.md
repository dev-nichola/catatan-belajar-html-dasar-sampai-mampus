# Input Image
Input type image itu jangan terkecoh dengan namanya walaupun nama typenya image input ini bukan digunakan berkas berupa gambar, melainkan digunakan untuk membuat tombol submit namun berupa gambar<br>
Input type button, submit atau reset menggunakan text untuk tombolnya, sedangakan image menggunakan gambar yang bisa kita ubah menggunakan atribut srd
